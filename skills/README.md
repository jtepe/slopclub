# Agent Skills

* [to-prd](./to-prd) **Create Product Requirements Document** Adapted from [Matt Pocock's skill](https://github.com/mattpocock/skills/tree/main) by the same name.
* [to-issues](./to-issues) **Create Issues From A Plan/PRD** Adapted from [Matt Pocock's skill](https://github.com/mattpocock/skills/tree/main) by the same name.
* [handoff](./handoff) **Handoff To Another Agent** Adapted from [Matt Pocock's skill](https://github.com/mattpocock/skills/tree/main) by the same name.
* [grill-with-docs](./grill-with-docs) **Grilling Session With Domain Knowledge** Adapted from [Matt Pocock's skill](https://github.com/mattpocock/skills/tree/main) by the same name.
* [do-issue](./do-issue) **Implement An Issue** Read an issue description and start to implement it.
* [pdf-reader](./pdf-reader) **Read PDF Files** Extract page-range text as markdown, figures by ID, and BM25-ranked keyword search over pages via the bundled `pdfread.py` CLI (needs a current [uv](https://docs.astral.sh/uv/)); includes a subagent workflow to keep context lean.
