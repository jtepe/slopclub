# Agent Skills

* [to-spec](./to-spec) **Create A Spec (aka PRD)** Adapted from [Matt Pocock's skill](https://github.com/mattpocock/skills/tree/main) by the same name (formerly `to-prd`).
* [to-tickets](./to-tickets) **Break A Plan/Spec Into Tickets** Adapted from [Matt Pocock's skill](https://github.com/mattpocock/skills/tree/main) by the same name (formerly `to-issues`).
* [wayfinder](./wayfinder) **Chart A Large Effort As A Map Of Investigation Tickets** Adapted from [Matt Pocock's skill](https://github.com/mattpocock/skills/tree/main) by the same name.
* [code-review](./code-review) **Two-Axis Review (Standards + Spec)** Adapted from [Matt Pocock's skill](https://github.com/mattpocock/skills/tree/main) by the same name.
* [tdd](./tdd) **Test-Driven Development** Adapted from [Matt Pocock's skill](https://github.com/mattpocock/skills/tree/main) by the same name.
* [handoff](./handoff) **Handoff To Another Agent** Adapted from [Matt Pocock's skill](https://github.com/mattpocock/skills/tree/main) by the same name.
* [grill-with-docs](./grill-with-docs) **Grilling Session With Domain Knowledge** Adapted from [Matt Pocock's skill](https://github.com/mattpocock/skills/tree/main) by the same name.
* [do-issue](./do-issue) **Implement A Ticket** Read a ticket description and start to implement it.
* [pdf-reader](./pdf-reader) **Read PDF Files** Extract page-range text as markdown, figures by ID, and BM25-ranked keyword search over pages via the bundled `pdfread.py` CLI (needs a current [uv](https://docs.astral.sh/uv/), or any Python 3.10+ with `pymupdf4llm` installed); includes a subagent workflow to keep context lean.
